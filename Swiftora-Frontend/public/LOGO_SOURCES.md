# Integration logos – where to get them

Save each file into this folder (`public/`) with the **exact filename** below so the site can load them.

---

## Channels

| Save as (exact name) | Official source |
|----------------------|-----------------|
| **Unicommerce-logo.png** | Wikimedia Commons (official-style logo): https://upload.wikimedia.org/wikipedia/commons/b/bd/Unicommerce_logo.png — or Unicommerce brand resources: https://info.unicommerce.com/ |
| **EasyEcom-logo.png** | From their site (right‑click logo → Save image), or contact: https://easyecom.io/ |

---

## Couriers

| Save as (exact name) | Official source |
|----------------------|-----------------|
| **Ekart** | Already in use: `ekart-logo.svg` (no file to add). |
| **Xpressbees-logo.png** | Official site: https://www.xpressbees.com/ — use the logo from the header/footer (right‑click → Save image as `Xpressbees-logo.png`). |

**Summary – only these files need to be added in `public/`:**

1. `Unicommerce-logo.png`
